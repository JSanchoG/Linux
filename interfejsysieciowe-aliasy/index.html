Konfiguracja interfejsów sieciowych w systemie Linux Ubuntu 18.04

Aby zacząć konfigurację interfejsu sieciowego, należy wykonać polecenie ifconfig, wyświetlą się interfejsy sieciowe, potem trzeba przejść do /etc/netplan używając polecenia cd. Jeżeli wystąpi błąd, konieczna będzie instalacja narzędzi sieciowych za pomocą polecenia sudo apt install net-tools

Następnie należy edytować plik 01-network-manager-all.yaml za pomocą polecenia sudo nano 01-network-manager-all.yaml

Można tu zmienić adres IP danej karty sieciowej, dodać bramę lub adresy DNS, należy pamiętać o wcięciach z reguły wynoszących dwie spacje
Aby zapisać edytowany plik, należy wciśnąć Ctrl+O, a następnie bez zmiany nazwy pliku wcisnąć Enter

Zapisać zmiany można wykonując polecenie sudo netplan apply

Odwzorowywanie adresów - aliasy sieciowe w systemie Linux Ubuntu 18.04

Aby dodać aliasy adresów w systemie Linux, należy wyedytować plik /etc/hosts

Można tu dodać dowolne aliasy dla adresów sieciowych

Poprawność wykonania można sprawdzić za pomocą polecenia ping
